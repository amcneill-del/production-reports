Executive Summary

This plan outlines a 90-day integration roadmap to unify 260 dental practice locations under a consistent digital standard. The work is organized into three sequential phases that build on each other: establishing the digital storefront foundation (websites, GMB profiles, social accounts), standing up scalable marketing performance reporting, and launching a 5-star patient experience program.

The plan leverages Gen4's existing Claude-powered audit toolkit — three core skills (dental-website-audit dental-gbp-audit marketing-performance) that automate the discovery, scoring, and reporting process at scale across both general practice and periodontal specialty locations.

Rather than attempting to audit, fix, and optimize all 260 locations simultaneously, the plan uses a pilot → scale approach: validate the process with SGA Priority Practices (Pony Express, Little Smiles, Peak Family Dental), then roll it out in waves of 25–30 locations per sprint while Ops maintains the running baseline.

🌐 1.1 — Website Audit & Baseline

▼

Run the dental-website-audit skill against all SGA Priority Practice websites first, then establish the rollout cadence for the remaining 260 locations. All sites live on WordPress or Webflow — audit findings will be categorized by platform to enable batched fixes.

Benchmark: 70% Website Audit Score

  • 50+ checkpoints across 10 categories (Conversion, UX, SEO, Services, Trust, NAP, Insurance/Financing, Content, Technical Performance, Tracking)
  • Each checkpoint scored as Pass / Partial / Fail / N/A
  • Score of 70% = strong baseline — most critical conversion and trust signals present
  • Locations below 70% enter a remediation queue with prioritized fix list

▼

Benchmark: 80% GBP Audit Score

  • Reviews, photos, and response quality carry the most weight for patient trust
  • Locations below 80% receive a prioritized optimization checklist for the OM

Open Decisions & Blockers

Items requiring leadership input before proceeding

1. Web Team Capacity for Remediation

Status: Web team has no capacity for large-scale site fixes.

Impact: Locations that fail the website audit below 70% cannot be remediated without a fix resource.

Options: (A) Outsource WordPress/Webflow fixes to an agency. (B) Hire a dedicated web resource for SGA. (C) Train Market Managers to make non-technical fixes (content, images, CTAs) and only escalate technical issues.

Decision needed by: End of Week 2

2. SGA Website Recommendations — What Do We Do With Them?

Status: SGA has its own website recommendations. Need to reconcile with Gen4 audit findings.

Impact: Conflicting or overlapping recommendations could slow execution.

Recommendation: Use the Gen4 dental-website-audit as the standard framework. Map SGA recommendations into the audit categories and flag any SGA-specific items that the audit doesn't cover.

Decision needed by: End of Week 3

3. Should We Run Peak Analysis (Marketing Performance) for Each Location?

Current recommendation: Not yet. Ops should focus on hitting the 70% website / 80% GMB benchmarks first. Full marketing-performance reports require practice management data that may not be accessible at scale yet.

Phased approach: Run reports for priority practices in Phase 2. Evaluate whether to scale to all 260 based on data availability and Ops bandwidth.

Decision needed by: End of Phase 1 (Week 4)

4. Paid Media Launch Timeline

Current status: Not launching paid media during the 90-day plan.

Prerequisite: Location must meet 70% website / 80% GMB benchmarks before paid spend is justified — otherwise you're paying to send traffic to a broken storefront.

Decision needed: When in Q3 2026 should we evaluate paid media readiness for top-performing locations?

5. Gen4 Playbook — What Changes for SGA?

Status: Current Gen4 version of the social/digital playbook exists. Need to determine what changes for the SGA context (260 locations, different brand mix, different Ops structure).

Decision needed by: End of Week 3

6. PRD Role Definition & Status

Status: PRD (Practice Relations Director) role needs clear definition for perio specialty practices. Also need a status update on where PRDs currently exist across the portfolio.

Impact: Without PRDs, referral program data collection and relationship management falls to someone else — who?

Decision needed by: End of Week 6

7. Amy's Training & Onboarding Path

Status: Amy (report author) needs a defined training curriculum to execute this plan at scale.

Key question: Who owns Amy's training? What does the progression look like — skills mastery → report generation → program facilitation → strategic planning?

Decision needed by: End of Week 9
